Vous pouvez utiliser cette section pour commencer immédiatement avec Avalonia. Cela vous guidera à travers un exemple de tutoriel simple, étape par étape, et vous présentera certains des concepts et techniques d'Avalonia.

Vous n'avez besoin d'aucune connaissance préalable d'Avalonia ou d'autres frameworks basés sur XAML pour apprendre de cette section.

Cliquez sur le bouton Suivant pour commencer.

:::info
Si vous êtes un développeur expérimenté dans l'utilisation de XAML et la création de projets Avalonia, vous voudrez peut-être passer à notre section de Bases [ici](../../basics) ou à l'un des Guides Pratiques [ici](../../guides).
:::

## Prérequis

### Installer les modèles

Avant de commencer, assurez-vous d'avoir [installé les modèles Avalonia](../install.md) :

<Tabs groupId="ide">
    <TabItem value="rider" label="Rider">
    [JetBrains Rider](https://www.jetbrains.com/lp/rider-avalonia/) fournit un support natif pour Avalonia XAML, offrant une expérience de développement riche sans nécessiter d'extensions supplémentaires.

    Pour des capacités d'aperçu XAML en temps réel, vous pouvez améliorer votre flux de travail en installant le plugin Avalonia Rider optionnel.

    <Button label="Télécharger le plugin Avalonia Rider" link="https://plugins.jetbrains.com/plugin/14839-avaloniarider" />

    </TabItem>
    <TabItem value="vs" label="Visual Studio">
    Les utilisateurs de Visual Studio 2022 peuvent améliorer leur expérience de développement Avalonia en installant notre extension officielle depuis le Visual Studio Marketplace.

    L'extension fournit une complétion de code XAML et un prévisualiseur en direct intégré, vous permettant de voir vos mises à jour d'interface utilisateur en temps réel.

    <Button label="Télécharger l'extension Avalonia pour Visual Studio 2022" link="https://marketplace.visualstudio.com/items?itemName=AvaloniaTeam.AvaloniaVS" />

    </TabItem>
    <TabItem value="vsc" label="Visual Studio Code">
    Bien que nous maintenions une extension [Visual Studio Code](https://code.visualstudio.com/) qui fournit un support XAML de base et un prévisualiseur en direct, nous recommandons d'utiliser [JetBrains Rider](https://www.jetbrains.com/lp/rider-avalonia/) pour la meilleure expérience de développement possible avec Avalonia.

    Cependant, si vous préférez continuer avec Visual Studio Code, vous pouvez installer notre extension depuis le marketplace.

    <Button label="Télécharger l'extension Avalonia pour Visual Studio Code" link="https://marketplace.visualstudio.com/items?itemName=AvaloniaTeam.AvaloniaVS" />

    </TabItem>
</Tabs>
